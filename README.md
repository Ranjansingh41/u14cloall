u14cloall
===================

Shippable CI image for clojure on Ubuntu 14.04. Lein2 version is available in this image.
Available clojure versions:

1. 1.3.0
2. 1.4.0
3. 1.5.1
4. 1.6.0
5. 1.7.0
6. 1.8.0
7. 1.9.0

**Services:**: Refer https://github.com/dry-dock/u14all for list of supported services
